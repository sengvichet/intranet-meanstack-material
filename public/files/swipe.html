Swipe

This UX pattern is intended for mobile devices only, and may not make sense to use on responsive sites. To initiate a swipe gesture on a desktop, you must click, hold and drag either right, left, up or down

Basic Usage

Swipe me to the left
Swipe me to the right
Swipe me up
Swipe me down 

 $scope.swipeDemo = {
    onSwipeLeft: function(ev) {
        alert('You swiped left!!');
    },

    onSwipeRight: function(ev) {
        alert('You swiped right!!');
    },
    onSwipeUp: function(ev) {
        alert('You swiped up!!');
    },

    onSwipeDown: function(ev) {
        alert('You swiped down!!');
    }
};
