Virtual Repeat

md-virtual-repeat specifies an element to repeat using virtual scrolling.

Basic Usage

Display 1000 item in a virtual-list with viewport of only 7 rows (height=40px).

This demo shows scroll and rendering performance gains when using md-virtual-repeat; achieved with the dynamic reuse of rows visible in the viewport area. Developers are required to explicitly use md-virtual-repeat-container as a wrapping parent container.

{{item}}
In HTML sample code angularjs scope variable denoted as {} instead of {{ }}. 

<div layout="row">
    <div flex="100">
        <md-virtual-repeat-container id="vertical-container">
            <div md-virtual-repeat="item in virtualRepeatDemoItems" class="repeated-item" flex>
                {item}
            </div>
        </md-virtual-repeat-container>
    </div>
</div>


$scope.virtualRepeatDemoItems = [];
for (var i = 0; i < 1000; i++) {
    $scope.virtualRepeatDemoItems.push(i);
}


#vertical-container {
    height: 292px;
    width: 100%;
    max-width: 400px;
    border: solid 1px grey;
}

#vertical-container .repeated-item {
    border-bottom: 1px solid #ddd;
    box-sizing: border-box;
    height: 40px;
    padding-top: 10px;
    padding-left: 10px;
}
