FAB Toolbar

You can use the fabToolbar with a trigger and regular toolbar.

Basic Usage

You may use the md-open attribute to programmatically control whether or not the control is open, and you may set the direction that the toolbar appears using the md-direction attribute. This component currently supports the left and right options.

md-fab-trigger inside icon should be use in md-icon attribute, otherwise after collapse fab toolbar trigger icon hide.
